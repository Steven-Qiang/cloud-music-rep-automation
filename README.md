# cloud-music-rep-automation
网易云音乐 顾问议事厅 自动化脚本。

顾问议事厅 https://music.163.com/st/repo/tag/mission

快速签到。随机判断。纯属刷积分。
### 仅供学习交流，严禁用于商业用途，下载后请于24小时内删除
## 使用

版本 **node**>12 依赖自装
将抓到的**cookie**存放到 **cookies.txt** 文件中。
接着运行```bash node app```
默认三十秒获取一次任务。自行修改代码。
